Dashboard Direction — Choose One

Three candidate layouts. Open each, compare how they handle: the 5-stage pipeline, the morning briefing, inline "WHY" explanations, and no-scrolling positions/scanner. Desktop 1920×1080+, dark mode, high-density "control room" style. None of these are production code yet.

Option 1
Pipeline HUD + Follow-Symbol
Persistent top-strip shows Scan→Eval→Order→Manage→Close funnel in real time. 3-column body (scanner / focus / stream + positions). Click any ticker anywhere → every panel filters to it.
BEST FOR: fleet-level situational awareness + drill-down on demand
Option 2
Unified Lifecycle Matrix
One massive dense grid. Rows = symbols, columns = the 5 lifecycle stages. Every stage of every symbol visible simultaneously. Bloomberg/options-chain density.
BEST FOR: seeing the whole pipeline state in one glance
Option 3
Decision Timeline Stream
Chronological event feed (like a Slack from the bot). Left sidebar = morning briefing. Center = stream of decision cards. Right sidebar = active positions with inline rationale.
BEST FOR: following the bot's narrative over time
WHAT TO JUDGE
  • • Can you see open positions WITHOUT scrolling?
  • • Does every number have an inline "why" explanation?
  • • Is the 5-stage pipeline always visible?
  • • Is there negative whitespace that feels wasted?
  • • Does the morning-briefing treatment match your mental model of "game plan"?
