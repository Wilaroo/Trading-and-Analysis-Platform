**Key Technical Concepts**
-   **Multi-Agent Architecture**: The new design paradigm for the AI system, involving a Router Agent for intent classification, and specialized agents for tasks like Trade Execution, Analysis, and Coaching.
-   **Hybrid AI/Code Safety Pattern**: A core principle where LLMs are restricted to language understanding (e.g., parsing intent from close TMC). All stateful information, numbers, and quantities are handled exclusively by deterministic code that interacts directly with data sources, preventing numerical hallucinations in critical operations.
-   **LLM Abstraction Layer**: An  component designed to decouple the agents from any specific LLM provider. This allows for hot-swapping between services like Ollama, OpenAI, and Anthropic without altering agent logic.
-   **MongoDB Persistence for Order Queue**: The trade execution pipeline was made robust against server restarts by migrating the order queue from a volatile in-memory dictionary to a persistent MongoDB collection.
