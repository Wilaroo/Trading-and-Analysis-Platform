**Key Technical Concepts**
-   **Learning Connectors Architecture**: A backend service design pattern to create explicit data feedback loops, enabling the system to learn from its own operations.
-   **Service-Oriented Backend**: A modular architecture where functionalities (data collection, learning sync, etc.) are encapsulated in distinct services that are initialized and registered in .
-   **Dynamic UI Configuration**: Building React UI components that fetch their state and configuration from the backend API, such as the Full Market button dynamically displaying the symbol count.
-   **Proxy-Based Local Integration**: The architecture for securely connecting a user's local service (Ollama) to the cloud-hosted backend via a proxy script () and a dedicated API endpoint.
-   **Leveraging Existing Infrastructure**: Reusing the robust 's symbol fetching and batching logic for the new  to avoid code duplication and improve reliability.
