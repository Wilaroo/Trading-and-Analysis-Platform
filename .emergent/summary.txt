-   **Chat-based Intent Detection**: Using regex and keyword matching within a service () to parse natural language commands and trigger specific backend actions (e.g., controlling the ).
-   **State Persistence for Services**: Saving the in-memory state of a long-running service (like the trading bot) to a database (MongoDB) on any state change and restoring it on initialization. This makes the service resilient to application restarts.
-   **Frontend State Synchronization with Polling**: Using  within a React component () to periodically fetch data from a backend API and update the parent component's state via callbacks, keeping the UI in sync with background processes.
